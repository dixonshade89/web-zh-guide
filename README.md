# web-zh-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容主要围绕中文互联网环境下的信息获取与使用指南，不针对任何单个域名或具体网站。

## 项目简介

web-zh-guide 是一个静态资源仓库，存放经过整理的 HTML 文档。每个页面均为独立文件，可直接在浏览器中打开查看，无需后端服务支持。

仓库内容聚焦于提供通用性、知识性的参考信息，帮助读者理解或使用某些网络资源。所有页面均保持其原始结构与风格，仅做必要的归档与发布处理。

## 目录说明

```
.
├── README.md           # 本文件
├── index.html          # 仓库入口页面（如有）
└── pages/              # 归档的 HTML 页面目录
    ├── page1.html
    ├── page2.html
    └── ...
```

- `pages/` 目录下存放所有归档的 HTML 文件，每个文件独立命名。
- 仓库根目录可能包含 `index.html` 作为导航页或说明页。

## 页面归档说明

- 每个 HTML 页面均为完整的独立文档，包含必要的样式与脚本（如有）。
- 页面内容可能引用外部资源（如图片、字体），但尽量保持自包含。
- 归档时可能对原始内容进行必要的格式调整或补充说明，但不会改变其核心信息。
- 所有页面均为静态内容，不包含动态交互或用户输入处理。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或修正已有内容。
- 如发现页面内容存在明显错误或失效信息，欢迎通过 Issues 提出。
- 不接受针对单个网站或服务的具体请求，仅接受通用性、知识性的归档建议。
- 本仓库不提供任何形式的保证或承诺，内容仅供参考。

## 使用方式

1. 直接访问 GitHub Pages 或其他静态托管服务（如已启用）查看页面。
2. 克隆仓库到本地，用浏览器打开 `pages/` 下的任意 HTML 文件。
3. 通过 `index.html` 浏览所有页面列表（如存在）。
